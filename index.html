Connect to GitHub

ClipSync uses a private GitHub Gist to sync between devices. One-time setup per device.

Step 1 — Create a token

Go to github.com/settings/tokens/new, give it any name, select gist scope only, and generate.

Saved only in this device's local storage. Never sent anywhere except GitHub's API.

Step 2 — Gist ID (optional)

Leave blank to auto-create a new private Gist. To sync with another device, paste its Gist ID here.

Settings

Update your GitHub token or Gist ID. Share the Gist ID with other devices to sync to the same queue.
