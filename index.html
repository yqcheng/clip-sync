Connect to GitHub Gist

ClipSync uses a private GitHub Gist to sync between your devices. You only need to do this once per device.

STEP 1 OF 2 — CREATE A TOKEN

Go to github.com/settings/tokens/new, give it any name, select gist scope only, and generate. Paste the token below.

Stored only in this device's localStorage. Never sent anywhere except GitHub's API.

STEP 2 OF 2 — GIST ID

Leave blank to auto-create a new private Gist. Or paste an existing Gist ID to share between devices.

Settings

Update your GitHub token or Gist ID. Changes apply immediately.
